# KanbanTable

Pattern:

```tsx
<KanbanTable>
    <KanbanTableHead>
        <KanbanTableRow>
            <KanbanTableTH>…</KanbanTableTH>
        </KanbanTableRow>
    </KanbanTableHead>
    <KanbanTableBody>
        <KanbanTableRow>
            <KanbanTableTD>…</KanbanTableTD>
        </KanbanTableRow>
    </KanbanTableBody>
    <KanbanTableFoot>
        <KanbanTableRow>
            <KanbanTableTH>…</KanbanTableTH>
        </KanbanTableRow>
    </KanbanTableFoot>
</KanbanTable>
```

Suffix slugs (kebab):

- kanban-table-head ↔ `<thead>`
- kanban-table-body ↔ `<tbody>`
- kanban-table-foot ↔ `<tfoot>`
- kanban-table-row ↔ `<tr>`
- kanban-table-th ↔ `<th>`
- kanban-table-td ↔ `<td>`
